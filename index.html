About Me

I’m a game programmer with experience in both Unreal Engine 5 and Unity, proficient in C++ and Blueprints for Unreal, as well as C# for Unity development. My work spans gameplay mechanics, UI systems, and integrations like APK builds, plugins, and cloud-based asset management. I enjoy tackling complex technical challenges, optimizing workflows, and ensuring smooth development across projects.

At Black Squirrels Studio, I’ve worked with Unreal Engine and Unity, contributing to multiple projects. In Unity, I’m implementing cloud asset management using the Addressables API and integrating the AdMob SDK for monetization. In Unreal Engine, I’ve worked on UI systems and developed a water and buoyancy system for our in-development game.

I’m also collaborating as an intern Gameplay and UI Programmer at Nuno4 Studio on Alien Guy, bringing gameplay features to life and refining UI elements to enhance player experience. My current tasks involve both UI and gameplay, specifically working on the creation of a map and a waypoint-based navigation system.

Whether working on core mechanics, UI, or backend integrations, I focus on writing clean, efficient code and ensuring seamless collaboration with artists, designers, and other developers.
